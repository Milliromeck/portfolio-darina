Портфолио 
Современный веб-сайт-портфолио на React с PHP-бэкендом для обработки контактной формы, развернутый на GitHub Pages.

Посмотреть сайт: https://milliromeck.github.io/portfolio-darina/

Как запустить:
1. Запуск всего через Docker (рекомендуется)
Если у тебя установлен Docker, можно запустить весь проект одной командой:
docker-compose up
2. Запустить только интерфейс (React)
если хочешь работать только с внешней частью сайта:
cd frontend
npm install
npm run dev
Сайт будет доступен по адресу: http://localhost:5173
